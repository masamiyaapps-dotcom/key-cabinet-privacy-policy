# key-cabinet-privacy-policy

Privacy Policy

Key Cabinet（以下「本アプリ」）は、ユーザーのプライバシーを尊重し、個人情報の保護に最大限配慮しています。

1. 収集する情報について

本アプリは、氏名、メールアドレス、位置情報、連絡先等の個人情報を一切収集しません。

本アプリに登録されるログイン情報（サイト名、ユーザー名、パスワード等）は、すべてユーザーの端末内にのみ保存されます。

2. 情報の利用目的

本アプリは、ユーザーが入力した情報を
アプリ内での表示・管理の目的のみに利用します。

3. 情報の外部送信について

本アプリは、外部サーバーへのデータ送信、クラウド同期、第三者への情報提供を行いません。

インターネット通信を利用することはありません。

4. 広告について

本アプリは、広告を表示しません。

5. アクセス解析について

本アプリは、アクセス解析ツールを使用していません。

6. セキュリティについて

本アプリは、Android OS が提供するセキュリティ機能および端末の認証機能を利用し、
ユーザー情報の保護に努めています。

7. プライバシーポリシーの変更

本ポリシーの内容は、必要に応じて変更されることがあります。
変更後のポリシーは、本ページにて公開します。
